What was the project?

I chose to combine these two projects to one page as they both document my javascript learning experience.

For the simple animation project I was tasked with building a website which included 15 buttons and different animations, styled with CSS, using javascript to control them.

For the project pool my group decided to create a todo list. The list allows you to add and mark off items.

What did I learn?

I learnt how to animate using CSS and control my buttons with Javascript.

I also learnt how to use javascript to control simpler lists.

What would I change today?

This module was the first time I had really worked with Javascript and it was difficult to wrap my head around. I still struggle with it, but I enjoyed making these simple projects.

As always I could continue working on the designs forever. I would have liked to design my own character for the simple animation project. If I had more time I would have also styled the to-do list.

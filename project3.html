02.01.02 Simple Animations

→ Link to my simple animation page

What was the project?

For the simple animation project I was tasked with building a website which included 15 buttons and different animations, styled with CSS, using javascript to control them.

What did I learn?

I learnt how to animate using CSS and control my buttons with Javascript.

What would I change today?

This module was the first time I had really worked with Javascript and it was difficult to wrap my head around. I still struggle with it, but I enjoyed making this simple project.

As always I could continue working on the design forever. I would have liked to design my own character for the simple animation project.
